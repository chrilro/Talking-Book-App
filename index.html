Create talking books for indigenous language documentation

A free, offline-capable tool for fieldwork linguists, community members, and students. Build printable, audio-linked storybooks from recordings, photographs, and transcriptions.

Open in English Abrir en Español
Features
📖

Story page editor

Add pages with transcription text and illustrations. Navigate, reorder, and edit any page at any time.

🗂

ELAN .eaf import

Select a tier and every annotation becomes a story page, ordered by start time. Works with any ELAN project.

🔊

QR audio link

Paste a Google Drive, Dropbox, or YouTube link. A scannable QR code is generated for the printed cover.

🖨

Print-ready output

Full-bleed cover with title, speaker, artist, project team, and QR code. One page per story page, school-ready.

📝

Markdown export

YAML front matter with all metadata, embedded base64 images, and a QR code rendered as an image for downstream processing.

🤝

Merge projects

Combine JSON files from multiple contributors with page-level conflict resolution — designed for fieldwork teams.

✈️

Works offline

A single HTML file. No installation, no server, no internet required. Open in any modern browser anywhere.

🌐

English & Spanish

Full interface available in both English and Spanish. All labels, dialogs, and exports are localized.

How to use

Click the link above to work in either English or Spanish.

Fill in the story details — title, language, speaker, artist, project team — in the left sidebar.

Paste a cloud audio link (Google Drive, Dropbox, or YouTube) to generate the cover QR code.

Add pages manually or import annotations from an ELAN .eaf file. Each annotation becomes one page.

Upload an illustration (.jpg or .png) for each page and enter or edit the transcription text.

Print the book — the browser print dialog produces a PDF with a full-bleed cover and one story page per sheet.

Save your project as a JSON file to continue later, share with collaborators, or merge with other contributors' work.
